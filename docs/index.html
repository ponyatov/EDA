Manual

Install

$ git clone -o gh https://github.com/ponyatov/EDA
$ cd EDA
$ make install

optional: rebuild docs/

$ cd EDA
$ make doc

Software packages

Software packages included in source-build distribution:

KiCAD

KiCAD is an Open Source electronics design CAD is well suited for offline PCB design on Linux workstation.

This project does not provide build scripts to run it on Windows, as this system is an unsuitable platform for the developer. Look first on the minimal M$ SDK install requires gigabytes of downloads, and this idiotic OS still has no package manager after more than 20 years of its history.
